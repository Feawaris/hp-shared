1. 快速上手

1.1. 安装

  • npm
  • cdn
  • tampermonkey

hp-shared (greasyfork.org)

  • pip
pip install hp-shared

1.2.使用

  • 简洁路径：已在 package.json 的 exports 和 miniprogram 字段配置，或根据环境配置 paths、alias、Import maps 等使用简洁路径，常用于 vue、node、小程序环境。
  • 完整路径：可自行选用 src 或 dist 目录，无法使用简洁路径的环境可使用完整路径，或根据上面配置简洁路径。
